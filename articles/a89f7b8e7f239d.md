# 1ヶ月目のまとめ

第二子誕生に合わせて4ヶ月の育休を取得し、最初の1ヶ月が経過しました。この1ヶ月を振り返っての大きな学びは以下の通りです。

- 事前のワンオペ経験が、いざというときの余裕につながる
- 役割分担をあらかじめ決めておくと、疲れている中でも動きやすい
- 上の子のケアは「妻の状態を見ながら柔軟に対応する」意識が大事
- 食事まわりのちょっとした外部サービスが、想像以上に助かる
- 読書はストレス軽減になる
- 家事を自動化するアイテムへの投資は、時間だけでなく心の余裕も生んでくれる

2ヶ月目以降も生活リズムは変化していくと思うので、引き続き記録を残していきたいと思います。以下は、この1ヶ月間の具体的な生活の記録です。

# はじめに

第一子のときは生後3ヶ月あたりからの育休だったこともあり、正直かなり余裕がありました。今回は生まれてすぐからのスタートだったため、想像以上にやることが多く大変だと感じています。

# 出産直後の1週間：ワンオペ生活

妻と第二子が入院している間、上の子を1人でずっと見る生活が1週間ほど続きました。
1人で子どもをずっと見るのはやはり大変でしたが、以前2泊3日のワンオペを経験していたことがしっかり活きて、なんとか乗り切ることができました。

# 退院後：夫婦で分担しながらの1ヶ月

退院後は、妻が元気に振る舞っていても「本当はそうではないはず」と自分に言い聞かせながら、まずは1週間、そして1ヶ月を一緒に乗り越えることを意識しました。

入院期間中は妻が先に第二子のお世話をしていたこともあり、私が同じように対応できるようになるまでは少し時間がかかりました。とはいえ第二子ということもあってか、飲み込み自体は早かったように思います。

## 分担の形

- 妻：食事の準備、夜間の授乳
- 私：掃除、保育園の送迎
- 就寝時の組み合わせ：私 + 第一子 / 妻 + 第二子

役割をある程度固定したことで、お互いに何をすべきか迷わずに動けるようになったのは良かった点です。

# 大変だったこと

## 食事の準備

1人目のときはなんとかなっていた食事の支度も、2人になると一気に大変さが増しました。特に昼ごはんは「楽できるものがあれば」と感じる場面が多く、Uber Eatsなどの宅配・ギフトをもらえると素直に嬉しいと感じました。

## 上の子のメンタルケア

第一子は基本的に私と一緒にいてくれますが、たまに「ママっ子」モードになる瞬間があります。そのときは妻の様子も見ながら、自分がフォローに回るかどうかをその都度判断するようにしていました。

## 自分たちのメンタルケア

空いた時間に、読書をしています。
イギリスのサセックス大学の研究によると、静かな場所でわずか6分間読書をするだけで、ストレスの68%が軽減されるとされています。これは音楽鑑賞や散歩よりも高い効果です。
心身のリラックスとして、読書に集中することで心拍数が低下し、筋肉の緊張がほぐれる効果があります。
